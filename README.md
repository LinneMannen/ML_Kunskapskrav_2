# Projekt struktur

Mnist_Modeller.ipynb : Mina modelleringar med motivering kring val och tillvägagångssätt
Teorifrågor.ipynb : Teorifrågorna i en notebook 
mnist_preprocessor_input.py : Skript för att processa bilder av handskrivna siffror till mnist liknande data.
mnist_svc_production : min färdiga modell sparad med joblib
streamlit_app.py : Mitt skript för streamlit appen för att predicta egna handritade siffror
siffra5 & siffra6 bilder på två siffror som jag testat lite med
requirements.txt : mina requirements till streamlit appen
självutvärdering : självutvärderingen enligt uppgiften
